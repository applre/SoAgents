# Scheduled Tasks Upgrade — Test Cases

> 对应实施计划：`docs/superpowers/plans/2026-03-25-scheduled-tasks-upgrade.md`
>
> 涉及功能：RunMode、EndConditions、AI Exit Tool、超时配置、执行计数

---

## TC-1: 新建任务 — 新字段完整展示

**前置条件：** 无

**步骤：**
1. 打开定时任务页面，点击「新建任务」
2. 填写名称 = "测试全字段"、Prompt = "报告当前时间"、工作目录 = 任意有效路径
3. 确认「运行模式」默认为「独立会话」
4. 切换运行模式到「持续会话」，确认 UI 切换正常
5. 输入「最大执行次数」= 3
6. 输入「截止时间」= 明天某时（点击 datetime picker 选择）
7. 勾选「AI 可自主退出」
8. 设置「超时」= 5
9. 提交

**预期结果：**
- [ ] 任务创建成功，跳转到 TaskDetail
- [ ] TaskDetail 显示运行模式 = 「持续会话」badge
- [ ] TaskDetail 显示已执行次数 = 0 次
- [ ] TaskDetail 显示超时限制 = 5 分钟
- [ ] TaskDetail 显示最多执行 = 3 次（已执行 0/3）
- [ ] TaskDetail 显示截止时间 = 所选时间
- [ ] TaskDetail 显示 AI 可自主退出 = 是
- [ ] TaskList 中该任务名称旁显示「持续会话」badge

## TC-2: 编辑任务 — 修改并清除新字段

**前置条件：** TC-1 创建的任务

**步骤：**
1. 在 TaskDetail 点击「编辑」
2. 将「最大执行次数」清空（留空）
3. 将「超时」清空（留空）
4. 取消勾选「AI 可自主退出」
5. 保存

**预期结果：**
- [ ] TaskDetail 中「最多执行」行消失（不再显示）
- [ ] 超时显示「10 分钟（默认）」
- [ ] AI 可自主退出显示「否」
- [ ] 验证字段确实被清除（不是保留旧值 3 / 5）

**预期结果：**
- [ ] AI 检测到文件存在后调用 `exit_scheduled_task` 工具

**清理：** 删除 `~/Desktop/done.txt`

**预期结果：**
- [ ] 任务被禁用（enabled = false）
- [ ] 下次执行时间清空
- [ ] 前端自动刷新任务状态
